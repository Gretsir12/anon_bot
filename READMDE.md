# 🤖 Анонимный бот с совладельцами

## Быстрый старт

### 1. Заполни `.env`
```
BOT_TOKEN=токен_от_BotFather
BOT_USERNAME=username_бота_без_@
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Запусти
```bash
python bot.py
```

---

## Возможности

### Для получателя (владелец ссылки)
| Действие | Описание |
|---|---|
| `/start` | Получить свою ссылку + кнопки управления |
| `/myid` | Узнать свой ID (для добавления в совладельцы) |
| Кнопка **Совладельцы** | Добавить/удалить соуправляющих |
| Кнопка **Сообщения** | Просмотреть входящие |
| Кнопка **Ответить** | Ответить на конкретное сообщение |

### Для совладельцев
Те же права что у владельца — видят входящие и могут отвечать.
Добавляются владельцем через кнопку **Совладельцы → Добавить** по Telegram ID.

### Для анонимных отправителей
- Переходят по ссылке → пишут сообщение
- Если им ответили — получают ответ в бот

## Структура БД (SQLite)

| Таблица | Что хранит |
|---|---|
| `users` | Все пользователи + уникальный token для ссылки |
| `co_owners` | Связь владелец → совладелец |
| `messages` | Сообщения (с реальными данными отправителя) |

---

## Как добавить совладельца

1. Второй человек пишет боту `/myid` — получает свой ID
2. Ты идёшь в `/start` → **Совладельцы** → **Добавить** → вводишь его ID
3. Готово — он получит уведомление и доступ ко всем твоим сообщениям
